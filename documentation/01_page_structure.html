F.A.Q

How do I install this template?

To install the tempalte, simply copy the items found inside the main archive ( the code folder ) to your server to your desired location.

How do I customize the contact form?

There is only one php file in the pack, simply edit it and replace the php file inside it with your address. The same php file is responsible for your homepage subscribe form.

How do I customzie the coach arrows? Why are they inline?

We decided to add the coach arrows to the html structure, for it's easier to have a controled classes by you, inside your page, rather than keep them in a CSS file with hundreds upon hundres of variants. To edit a coach arrow, check the coach folder in the images, and use the arrow class like the image name. Check our provided example.

Page Navigation

The page navigation is inserted in the page-sidebar-scroll class, and looks like this:

        <div class="page-sidebar-scroll">
            <a href="#" class="nav-icon nav-coach coach-icon">COACH</a>
            <a href="index.html" class="nav-icon home-icon-selected">HOME</a>
            <a href="type.html" class="nav-icon type-icon">TYPE</a>
            <a href="jquery.html" class="nav-icon jquery-icon">jQUERY</a>
            <a href="others.html" class="nav-icon other-icon">OTHER</a>
            <a href="gallery.html" class="nav-icon gallery-icon">GALLERY</a>
            <a href="folio.html" class="nav-icon folio-icon">FOLIO</a>
            <a href="blog.html" class="nav-icon blog-icon">BLOG</a>
            <a href="mail.html" class="nav-icon contact-icon">MAIL</a>
            <a href="tel:+1 234 567 890" class="nav-icon call-icon">CALL</a>
            <a href="sms:+1 234 567 890" class="nav-icon text-icon">SMS</a>
        </div>

Every class has a selected version and an unselected version, to add a selected value to an icon, simply add -selected at the end of it.

To create more icons, we've provided a PSD file in the design icons! Use the icon pack we provide in the images/icon folder, to add more icons.

Containers and page elements.

Page elements are added within the second part of the page structure, in the content class. Every item you add in this part of the page, except for the image slider, must be housed in a <div class="container"></div>

Please check the second documentation for a better understanding of container ussage and also, to see what features our framework provides!
